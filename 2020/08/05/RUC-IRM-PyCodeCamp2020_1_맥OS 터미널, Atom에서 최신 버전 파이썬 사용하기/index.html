맥OS 터미널, Atom에서 최신 버전 파이썬 사용하기

안녕하세요, 괜찮은 괜찮은입니다.

맥에는 기본적으로 파이썬이 깔려있지만, 이 디폴트 파이썬이 구버전인 2.7버전이어서 머지 않아 지원이 중단된다는 경고창이 뜨곤 하죠.
문제는 최신 버전인 파이썬3을 설치해도 터미널에서 python버전 체크를 했을 때 계속 파이썬 2.7로 나오고 실행되는 것인데요,
이 상태에서는 많은 분들이 사용하고 계시는 Atom 에디터에서 script 패키지를 설치해서 파이썬을 연동시켜봐도최신 버전 파이썬이 아닌 파이썬 2.7 버전으로 구동이 되어 Errno 2 No such file or directory 에러가 발생합니다.
이번 글에서는 python을 쉘에서 입력했을 때 기본값(디폴트값)으로 설정된 2.7버전을 실행하는 대신 현재 보편적으로 사용하는 파이썬3을 실행하게 하는 방법을 알아보겠습니다.
(터미널 쉘이 zsh가 아닌 분들은 아래 글의 1번까지 진행해 bash쉘에서 zsh쉘로 바꿔주세요!)

Mac bash 대신 zsh 사용하기

(macOS 카탈리나(2019년 10월이후 모델 혹은 이 이후 업데이트) 부터는 터미널 기본 shell 환경이 zsh입니다!)



0.

Atom을 실행해 아무 파일이나 들어가서(혹은 아무 코드나 쳐서) 터미널로 실행해, Errno 2 No such file or directory 의 형식을 확인합니다.

1.

cmd + space bar 에서 터미널 검색 / 트랙패드에서 네 손가락을 동시에 오므려 Lanchpad 실행 > 기타 > 터미널의 경로를 통해 터미널을 엽니다.


2.

(homebrew가 설치되지 않은 경우)

홈브루 한국어 페이지

와 같이

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

을 복사해서 터미널에 붙여넣기하고 엔터를 입력해 실행시켜주세요.

(시간이 약간 소요되며, 중간에 두어 번 엔터를 눌러줘야 합니다.)
설치가 끝난 뒤, 차례로
brew install tree

brew install wget
brew upgrade

를 입력한 뒤 엔터를 눌러 설치해주세요.


3.

최신 버전의 Python을 설치하기 위해 터미널에 아래의 문장을 입력해 다운로드해줍니다. (공홈에서 다운받으셔도 무방하지만 경로가 달라져서 스크린샷과 다를 수 있으니, 헷갈리기 싫으신 분들은 터미널로 진행해주세요. )

brew install python

설치가 완료된 후에는

python -V

를 입력해 디폴트 파이썬 버전을 확인해주세요.
(2.7버전이며 사실 이 부분은 건너 뛰어도 무방합니다.)
이제
which python

which python3
을 차례로 한 줄씩 입력해서 경로를 확인하세요.



4.

터미널에 아래 문장을 입력해 최신 버전의 pyenv를 설치해주세요. 설치가 완료되면 버전 정보가 나옵니다.(최신 버전의 파이썬과 같은 숫자일겁니다)
brew install pyenv

그 다음에는 아래의 문장을 입력해 마저 설치를 진행해 주세요. pynev install 3.8.5

이제 위의 Installed Python-3.8.5 to /Users/… 문장이 출력됐으면반은 끝난 겁니다.
지금은 최신 버전의 파이썬이 pynev를 통해 설치된 상태이며, 이제 우린 pynev를 통해 파이썬3를 글로벌 디폴트로 설정할겁니다.
pynev global 3.8.5

pynev version
를 각각 입력해

3.8.5 (set by /Users/사용자명/.pyenv/version)
이 출력되나 확인하세요. (2020-07-31 기준)

이어서

echo -e 'if command -v pyenv 1>/dev/null 2>&1; then\n eval "$(pyenv init -)"\nfi' >> ~/.zshrc

를 입력해주세요. (이건 입력하고 줄만 바뀌고 끝나니 안 불안해하셔도 됩니다.)
vi ~/.zshrc
를 입력해 zsh쉘에

위와 같이
`if command -v pyenv 1>/dev/null 2>&1; then

eval “$(pyenv init -)”
fi`

의 내용이 있는지 확인하시고, 없을 시엔 복사 붙여놓기 해줍니다.

PATH="/System/Library/Frameworks/Python.framework/Versions/3.8/bin:${PATH}" export PATH

는 없을테니 fi 바로 밑줄에 복사해서 붙여넣어주세요.
0번의 구버전(2.7) 경로를 2020-07-31 기준 최신버전 파이썬 저장 경로로 고친 경로입니다.
이후엔
‘enter’ >’Esc’ > ‘:’ > ‘w’ > ‘q’ >’enter’
를 차례로 입력해 zsh를 나와주세요.
그리고 난 뒤엔 터미널을 끄시고 새로운 터미널 창을 열어주세요.(아톰도 종료시켜주시구요!)

5.

which python
python -V

를 입력해 새로 설정된 위치와 버전정보를 확인하세요.

Python3.8.5 저장 경로와 python3.8.5가 뜹니다! :)



6.

(5번까지 실행했는데도 안되었으면 진행해주세요.)

Atom을 열어 Packages에서 아마 깔려있을(없으면 설치해주세요)script package를 선택하고 script package의 Settings로 들어가주세요.

그리고 왼쪽 밑의 ‘Open Config Folder’를 실행시켜줍니다.
위 packages/script/grammars/python.coffee 경로로 들어가서
위 스크린샷과 같이 python.coffee 문서의 Selection Based 부분의 커맨드가 ‘python’으로 된 두 부분을 전부 ‘python3’로 바꿔주고 저장하고 Atom을 재시작해주세요.
