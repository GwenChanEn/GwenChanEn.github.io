书面作业 P84 第 1、2 题

（1）布拉德福定律的产生背景是什么?该定律是怎样形成的?

  • 布拉德富定律产生的客观背景：

    1. ​ 进行科学研究和文献工作时，布拉德富感到了科学文献的分散是显而易见的普遍·客观现象。他还发现该现象的带有规律性，该规律理论上可以从科学统一性原则出发定性地推导出来，也可以从对相关期刊所载论文的数量统计基础上推导出定量的结果。

    2. ​ Bradford认为，之所以有一个科学文献出现在另一个学科的期刊之中这种现象，是因为’科学统一性原则‘——科学技术的每一个学科或多或少（或远或近）地与其他任何一个学科相关联。他想知道科学与学科的相互关系如何影响文献、其比例以及分散特点，通过一系列研究得出结论：对一个专家有用的论文，不仅仅会出现在这个专家所在科学的专业期刊上，而且也时时可能出现其他学科的期刊上。这些“其他学科”期刊的数量，随着它们的研究领域与那个“专家所在”学科的关系密切程度的变小，以及关于那个“专家所在”学科的论文在每种期刊中的登载率的减少而变大，这呈一种反比例关系。但是，这种关系密切与否不仅有着相对关系，而且有着多个衡量等级关系。各种期刊、各个学科的情况都不一样，决定如何分散的收众多因素的影响。

      ​ 加上，布拉德福还发现总有某种期刊的内容与某个学科更近些；反而总有某种数量更多的期刊的内容离这个学科更远些；论述某学科的论文数量一定要多于论述有关学科的论文。这几个道理后续为将期刊划分为几个区域的思想作为基础。（按照期刊登载有关某个学科论文的载文率的高低来划分区域，每个区域中的期刊数量随着该区域期刊载文率的减少而增多）

    3. ​ 一些文献统计研究为布氏[shì]定律做了一种奠基。1930年代具有存储、报道、检索文献职能的文摘杂志不断出现，而且其增长很快。但是，这些文献的重复·遗漏现象屡次出现，让布拉德福思考以及探索这些不完整现象和文献的分散现象之间的关系。

  • 布拉德富定律形成过程

    他将期刊按照相关论文载文量的多少以渐减的顺序排列起来，然后采取三种不同的方法进行了分析：

    1）区域分析

    ​ 他按照年平均载文量的多少，把应用地球物理学和润滑的期刊都分成三个区：a，四篇以上的期刊；b，多于一篇而少于四篇的期刊；c，一篇或不足一篇的期刊。经过分析，他认为每个学科期刊的各区相关论文数大体相等，而相继各区的期刊数基本上成等比数列，公比近似等于5。

    2）图像观察

    ​ 布拉德福以横轴表示期刊累积数的对数、纵轴表示相关论文累积数，描绘了这两个学科的相关论文分布的曲线。他观察这一图像后认为，若略去下部分的弯曲部分，则是一条直线；并由此出发，得出了前一、二、三个区的期刊数量也是成等比增加的结论。

    3）数学推导

나무위키_중국유학

+입시

한국초중고등학교를 졸업하고 한국에서 대학까지 다녀본 입장으로서 느낀 바를 서술해보자면, 전체적인 유학생 분위기는 인문적인 소양이나 학풍을 추구하는 것 보단 국내외 여러 대학과 비교당하는 열등감을 이겨내보려는 느낌이 없지않아 있습니다.

입시에서 수학의 경우엔 문이과 구분이 없어 이과수학까지 다 해야 하지만 어려워 봤자 4점 중등 난이도이니 (킬러문항정도의 난이도는 북경대에서 몇 년에 한 번 맨 마지막 한 문제 정도..?) 학문적인 깊이로는 한국 수능으로 한양대 갈만큼 어렵다고 생각하진 않습니다. 하지만 중국어 원어로 다른 중국인과 똑같이 대학 수준 수업을 들을 수 있는 중국어 능력을 갖춘 사람의 수는 그보다 훨씬 적을거라 단언합니다.

북경대 19학번 笔试 합격자 200명 중에 화교가 130명이라는 썰이 있습니다. 사실일 가능성이 높으며 실제로 다는 아니지만 중국에서 친척들까지 오래 살거나 너무 어릴 적부터 산(1n년?) 한국국적 유학생들이랑 얘기하다보면 간혹 사상이 불쑥불쑥 튀어나와서 흠칫할 때가 종종 있는데요. 공산당이라는 존재와 그 영향력에 대해 정말 생생하게 알고 있는 친구들이므로 무조건적으로 배척하는건 좀 아닌 것 같고 만일 한국에서 오래 사신 분이 중국대학 입시를 준비하시려고 마음먹는다면 이 친구들이랑 경쟁하게 된다고 생각하시면 됩니다.

국내 모 대학에서 언어전공을 하다 차라리 해외 대학 본과 좋은 학점으로 졸업해서 외국어와 전공 당장 당장 실무 투입할 수 있다고 기업에게 인정받는게 낫지 않을까 하는 생각에 학교를 무턱대고 바꾸자고 생각하고 적당히 탱자탱자 놀다가 모 입시학원에 반년정도 있었는데요, 외국어는 국내에서 배우는 것보다 외국에서 배우는 것이 훨씬 경쟁력 있다고 생각합니다.

국내에서 고대 경희대 중문과 나와도 애초부터 특례나 언어특기자로 대학간 케이스가 아닌 한 중국에서 1-2년 유학한 중학생보다 못하는 경우 수두룩합니다. 외고 나와도 저희나 중국인 기준이지만 잘 못하는 건 당연하고 솔직히.. 6급 간신히 따고 되게 잘하는 줄 아는 사람 많습니다. (물론 저도 그랬구요^^ 그치만 적어도 항공항천대 전액 장학생·상교대~ 비대외한어과 본과 유학생들이 보기엔 약간 우스울 수도 있습니다.)

    NumPy’s array class is called ndarray. It is also known by the alias array. Note that numpy.array is not the same as the Standard Python Library class array.array, which only handles one-dimensional arrays and offers less functionality. The more important attributes of an ndarray object are:

    • ndarray.ndim ]

      the number of axes (dimensions) of the array.

    • ndarray.shape ]

      the dimensions of the array. This is a tuple of integers indicating the size of the array in each dimension. For a matrix with n rows and m columns, shape will be (n,m). The length of the shape tuple is therefore the number of axes, ndim.

    • ndarray.size ]

      the total number of elements of the array. This is equal to the product of the elements of shape.

    • ndarray.dtype ]

      an object describing the type of the elements in the array. One can create or specify dtype’s using standard Python types. Additionally NumPy provides types of its own. numpy.int32, numpy.int16, and numpy.float64 are some examples.

    • ndarray.itemsize ]

      the size in bytes of each element of the array. For example, an array of elements of type float64 has itemsize 8 (=64/8), while one of type complex32 has itemsize 4 (=32/8). It is equivalent to ndarray.dtype.itemsize.

    • ndarray.data ]

      the buffer containing the actual elements of the array. Normally, we won’t need to use this attribute because we will access the elements in an array using indexing facilities.
